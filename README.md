# ARIES Data Archive

This repository contains scripts, utilities, and workflow components developed for the ARIES Data Archive project. The purpose of this project is to process, organize, validate, and prepare astronomical observational data for structured archival storage and future scientific use. It supports efficient handling of data products and metadata from ARIES facilities, helping improve consistency, accessibility, and long-term preservation.

## Scientific and Technical Relevance

Astronomical archives are essential for preserving observational datasets and enabling reproducible scientific research. Properly curated data archives allow future users to access, interpret, and reuse telescope data efficiently. This project contributes to the infrastructure required for reliable archive development and management at ARIES.

## Objectives

- Organize observational data into archive-ready structures
- Extract and validate metadata from FITS files
- Automate data handling and reporting workflows
- Improve consistency and traceability of archived datasets
- Support long-term scientific usability of ARIES observations

## Scope

This repository includes tools for:
- FITS header extraction
- metadata validation
- file organization
- summary table generation
- logging and reporting
- archive preparation workflows
